# Marketing-Performance-Dashboard

Interactive Excel dashboard for analyzing marketing campaign performance and KPIs.

## Tools Used
- Excel
- Pivot Tables
- Power Query

## Features
- Campaign performance analysis
- KPI tracking and reporting
- Engagement and reach analysis
- Interactive pivot tables and dashboards
- Data cleaning and transformation

## Key Insights
- Analyzed campaign engagement and estimated reach
- Identified top-performing marketing campaigns
- Tracked sales and engagement KPIs
